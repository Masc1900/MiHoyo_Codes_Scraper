How to Redeem HSR Codes

Use the Official Redeem Code Website

  1. Go to the official redemption site.
  2. Log into your account.
  3. Select your account's region.
  4. Input the redeem code and click ''Redeem''.
  5. Claim rewards from your in-game mail.

Players can redeem codes on the game's official redemption website. Just log in with your HoYoverse account and enter your server, character name, and valid redeem code to claim the free rewards.

Note that for IOS users, using the website is the only way to redeem codes!

Star Rail Redeem Code Website

In-Game Redemption

  1. Open the menu and click on the ''ellipsis'' icon
  2. Click on the ''Redemption Code'' button
  3. Input your Redeem Code and click ''Redeem''
  4. Go back to ''More'' and claim rewards from the Mail
  5. Return to the Menu then click on the Mail Icon
  6. Click the Claim All button to get the rewards.

It is also possible to redeem codes in-game by opening the Menu, then tapping on the ellipsis in the top right corner. Select the Redemption Code option and enter the codes.

Note that the Redemption Code button will not show up on the IOS version of the game!

How to Reedem Codes on Mobile

  1. Tap the Phone Icon to bring up the menu
  2. Tap on the ''ellipsis'' icon
  3. Tap on the ''Redemption Code'' button
  4. Input your Redeem Code and tap ''Redeem''
  5. Return to the Menu then tap on the Mail Icon
  6. Tap on the Claim All button to get the rewards.

Players on Mobile can open the Menu by pressing the Phone icon at the top left of the screen, near the mini-map. Then, tap the ellipsis at the top right as mentioned previously to open the Redemption Code function!

What are Redeem Codes?

Rewards Stellar Jades, Materials, and Credits


Redeeming Honkai: Star Rail codes while they're valid will reward you with different items, including Stellar Jades, Consumables, Upgrade Materials, and Credits.

Rewards obtained from redeem codes can be claimed in-game through your mailbox. The mailbox will be unlocked during the mission, A Moment of Peace.
